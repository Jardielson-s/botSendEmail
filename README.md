# MicroService

## Pré-requisitos

* typescript
* rabbitmq
* express
* mongodb e mongoose

## Objetivos

O objetivo do projeto é a construção de um microserviço para envio de email.

## O que a aplicação pode fazer

- [ ] microServiço para enviar email
- [ ] listar email
- [ ] verificar se email foi lido
- [ ] enviar email para mais de uma pessoa
